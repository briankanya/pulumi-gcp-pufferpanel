============
Installation
============


Stable release
--------------

To install pulumi-gcp-pufferpanel, run this command in your terminal:

.. code-block:: console

    $ pip install pulumi_gcp_pufferpanel

This is the preferred method to install pulumi-gcp-pufferpanel, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for pulumi-gcp-pufferpanel can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/briankanya/pulumi_gcp_pufferpanel

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/briankanya/pulumi_gcp_pufferpanel/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/briankanya/pulumi_gcp_pufferpanel
.. _tarball: https://github.com/briankanya/pulumi_gcp_pufferpanel/tarball/master
